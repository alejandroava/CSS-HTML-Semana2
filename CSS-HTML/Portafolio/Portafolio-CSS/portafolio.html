Sitio web para estudio musical

Mi primer proyecto lo realize para un estudio de musica llamado Doble m music utilizandola plataforma wordpres.La pagina web cuenta con la visualizacion de sus servicios de grabacion , mezcla y composicion y una presentacion del studio con la ubicacion en un mapa.Tambien se creo una pagina para reproduccir canciones y dirigir a los clientes a las plataformas digitales de musica como spotify, deezer y aple music como las principales. se creo una pagina aparte con link de Youtube donde los usuarios pueden ver los videos musicales desde el mismo sitio web.Otra especificacion fue crear una pagina donde aparecieran todos los artistas con una foto y link de direccionamiento a las redes sociales de cada artista. se implemento un diseño acorde a los colores representativos de la marca y formulario para pedir citas online.

Pagina de noticias

En este proyecto cree una pagina de noticias con solo HTML utilizando el editor de codigo visual studio code. la pagina es sobre tecnologia donde se diseño en la parte superior un menu de navegacion con tablas para visualizar el las noticias con autor, fecha y titulo de la noticias. la tabla tiene para acceder por medio de un clik a cada noticia para facilitar al usuario la navegacion y busqueda de lo que quiere ver. se aplicaron listas con link externos para complmementar la informacion y videos dentro de la pagina para que el usuario pueda verlos sin nesecisidad de salir de la misma.
